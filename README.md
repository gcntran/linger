# linger
A web-based cozy game developed with Phaser 3.
Genre: 2d, pixel art, exploration, narrative, slice-of-life 

## How to Install & Run
### For Players (The Easy Way)
1. **Download** the project zip file from GitHub or itch.io: https://gcntran.itch.io/linger 
2. **Extract** the folder to your computer.
3. **Open** `index.html` in any modern web browser.
   * *Note: Some browsers block local files. If the game doesn't load, use the "Developer" method below.*

### For Developers (Local Server)
Because this game uses external assets, browsers may block them due to "CORS" security. It is best to run it using a local server:
1. Ensure you have [VS Code](https://code.visualstudio.com/) installed.
2. Install the **Live Server** extension.
3. Open the project folder in VS Code.
4. Click **"Go Live"** in the bottom right corner of the window.

### Standard Method (npm)
1. **Install dependencies:**
   ```bash
   npm install
2. **Run in development mode:**
    ```bash
   npm run dev

## How to Play
- **Explore:** Use WASD to move Rem around the house.
- **Interact:** Click on objects when the cursor changes to uncover hidden Tarot cards.
- **Get Help:** Click on **Dot** (the cat) at any time for hints on where to look next.
- **Interface:** Use the **?** icon in the top corner to toggle the help overlay.

## Features
- **Atmospheric Storytelling:** A "cozy-core" aesthetic inspired by Stardew Valley.
- **Intuitive UX:** Context-aware cursor changes and a dynamic hint system.
- **Polished Visuals:** Hand-crafted pixel art with smooth WebGL transitions.

## Technical Stack
- **Engine:** Phaser 3 (HTML5/JavaScript)
- **Design:** Figma & Aseprite
- **Resolution:** 1920x1080 (Optimized for desktop and itch.io)
